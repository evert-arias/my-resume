# My Resume

Resume generator using [JSON Resume](https://jsonresume.org/) and theme from [Autumn](https://themes.jsonresume.org/theme/autumn) by [Lorenzo Dal'Aqua](https://github.com/lorenzodalaqua)



# Install Dependencies

`npm install`

# Serve

`npm run serve`

# Export HTML

`npm run export:html`

# Export PDF

`npm run export:pdf`
